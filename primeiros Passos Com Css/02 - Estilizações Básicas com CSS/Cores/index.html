Cores

* Cores Pré-Definidas

Existem várias formas de definirmos cores utilizando o CSS. Elas são:

  • Cores pré-definidas
  • currentcolor
  • Hexadecimal
  • Hexadecimal com transparência
  • RGB
  • RGBA
  • HSL
  • HSLA

* HSL

hsl(hue, saturation, lightness)
  • Hue(matiz): é o grau da roda de cores (de 0 a 360 graus)
    • 0 ou 350 é vermelho
    • 120 é verde
    • 240 é azul
  • Saturation(saturação): valor percentual
    • 0% um tom de cinza
    • 100% cor total
  • Lightness(luminosidade): luminosidade da cor
    • 0% é preto
    • 100% é branco
