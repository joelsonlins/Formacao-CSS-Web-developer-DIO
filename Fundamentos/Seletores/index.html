* Seletor de Tag/Tipo

Bem-Vindo!

No CSS nós possímos os seletores por ID

  • Seletor por tipo: Busca elementor por uma tag HTML específica
  • Seletor por ID (#): Busca elementos através do atributo "id"
  • Seletor por Classe (.); Busca elementos através do atributo "class"
  • Seletor Universal (*): Seleciona todos os elementos HTML
  • Seletor de Atributo ([atrib]): Seleciona elementos que possuem
    um atributo específico em nosso documento HTML e conseguimos
    também buscar atributos com valor específico
