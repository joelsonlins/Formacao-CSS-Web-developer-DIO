Propriedades Letter-Spacing e Word-Spacing

Letter-Spacing

Adiciona um espaçamento entre cada CARACTERE do seu texto. Como o nome já diz, espaçamento entre as letras.

Word-Spacing

Adicionar um espaçamento entre cada PALAVRA do seu texto. Como o nome já diz, espaçamento entre as letras.

Propriedade Word-Wrap

Define, caso uma palavra seja muito grande para caber em uma linha, se ela será quebrada em qualquer caractere para que uma parte dela fique na linha e o restante na próxima, ou se não, toda palavra deve ficar na mesma linha e ultrapassar os limites ue definimos para o elemento.

word-wrap: normal;
