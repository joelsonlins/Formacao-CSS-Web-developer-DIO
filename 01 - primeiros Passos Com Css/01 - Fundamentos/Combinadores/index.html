* Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos
separar os seletores utilizando uma virgula (,).

Assim, não precisamos ficar copiando e colando os mesmo códigos para todos, nós podemos
fazer um só bloco de CSS e reutilizar as mesmas regras!
